ElectronicSeeds Application is an iOS (mainly for iPhone) that users can share information very easily.

    - pod 'AFNetworking', '~> 2.5'
    - pod 'FXForms', '~> 1.2.13'
    - pod 'Dropbox-iOS-SDK', '~> 1.3.13'
    - pod 'GoogleMaps', '~> 1.10.2'
    - pod 'SVProgressHUD', '~> 1.1.3'
